* JavaScript在设计之初，为了方便初学者学习，并不强制要求用var申明变量。这个设计错误带来了严重的后果：如果一个变量没有通过var申明就被使用，那么该变量就自动被申明为全局变量启用strict模式的方法是在JavaScript代码的第一行写上：`'use strict'`; 这样会强制使用var声明变量
